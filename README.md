# Calorie Tracker (MVP)

Full-stack calorie and activity tracking application, that helps users calculate there daily target calories to lose weight , log food and exercise.

## Status
MVP in progress.

## Roadmap 
-[x] Authentication (backend + frontend)
-[x] Homepage UI
-[x] Signup and login page
-[ ] Get user body stats
-[ ] Calories tracking dashboard
-[ ] weekly charts


## Tech Stack
### Frontend
-React (Vite)
-React Router
-Fetch API
-CSS modules / custom css

### Backend 
-Nodes.js
-Express
-MongoDB + mongoose
-JWT authentication
-bcrypt


## Features
-User authentication (signup & login)
-JWT based auth 
-Responsive homepage UI
-Login and signup pages with validation
-Calorie Tracking dashboard in progress


## Getting started
### Backend 
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install 
npm run dev
