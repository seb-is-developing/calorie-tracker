# Calorie Tracker (MVP)

Full-stack calorie and activity tracking application, that helps users calculate there daily target calories to lose weight , log food and exercise.

## Status

MVP in progress. - core authentication and calorie tracking features implemented

## Roadmap

-[x] Authentication (backend + frontend) -[x] Homepage UI -[x] Signup and login page -[x] Get user body stats -[X]Exercise Tracking -[X]Food Tracking -[x] Calories tracking dashboard -[x] Weekly charts

## Tech Stack

### Frontend
- React (Vite)
- React Router
- Fetch API
-CSS modules / custom css

### Backend
- Nodes.js
- Express
- MongoDB + mongoose
- JWT authentication
- bcrypt

## Features
- User authentication (signup & login) using JWT
- Responsive homepage UI
- Form Validation for log in and singup
- Calorie Tracking dashboard in progress
- Users can input body stats (age , gender, height , weight , amount of exercise)
- Daily Calorie target for weight loss calculation on first signup
- Daily consumption tracking
- Weekly history view for past Calorie totals

## Getting started
### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev
